Resume Builder API

ASP.NET Core Web API for Resume & Cover Letter Creation

Overview Architecture Authentication Endpoints Models Examples Support

Overview

The Resume Builder API allows developers to generate professional resumes and cover letters based on user information and job descriptions. It integrates with OpenAI for AI-driven content, stores structured data in PostgreSQL, and exposes endpoints for user and job management.

Base URL

https://api.resumebuilder.com/v1

  3. Keep your Pass Code secure.

Endpoints

UserController

  • POST /User/register – Register new user
  • PUT /User/update – Update existing user
  • DELETE /User/delete/{publicId} – Delete user
  • GET /User/getUser/{publicId} – Get user details

JobController

  • GET /Job/fetch-rcl-json – Fetch Resume & Cover Letter JSON
  • GET /Job/get-matchKeys – Get match percentage & key skills
  • GET /Job/get-all – Full job analysis (resume + cover letter + skills)

Code Examples

Quick examples of calling the API.

# curl example
curl -X POST https://api.resumebuilder.com/v1/User/register \
  -H "Content-Type: application/json" \
  -H "x-passcode: YOUR_PASSCODE" \
  -d '{"firstName":"John","lastName":"Doe","email":"john@example.com"}'
