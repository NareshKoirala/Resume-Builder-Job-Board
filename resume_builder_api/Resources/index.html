API Overview

The Resume Builder API, developed using ASP.NET Core, allows developers to programmatically create, manage, and export professional resumes and cover letters. It enables customization of resumes and cover letters based on job details provided by the user. The API uses user data to generate tailored resumes and cover letters for specific job applications. Additionally, it provides a job-matching percentage to indicate how well the user‚Äôs profile aligns with the job requirements and identifies key skills the user needs to develop to improve their suitability for the role.

Resume/Cover Letter Creation

Build resumes and cover letters with customizable sections like personal info, experience, and skills.

Template Management

Choose from a variety of resume and cover letter templates tailored to different industries.

Export Options

Export resumes and cover letters as PDF, Word, or HTML with configurable styling.

User Management

Manage user profiles and their associated resumes.

Base URL

https://api.resumebuilder.com/v1

Response Format

{
  "success": true,
  "data": {...},
  "message": "Operation completed successfully",
  "timestamp": "2025-08-11T01:28:00Z"
}

Authentication

The API requires API key authentication. Include your key in the request headers:

Authorization: Bearer YOUR_API_KEY
Content-Type: application/json

Getting Started

  1. Register at the developer portal.

API Endpoints

The API is organized into controllers for users, resumes, and templates. Below are the available endpoints.

UsersController

Endpoint Method Description Parameters
/users/{userId}/resumes GET Retrieve a list of resumes for a user
  • userId (path) - Unique user ID (string)
  • limit (query) - Max results (integer, default: 10)
  • offset (query) - Results to skip (integer, default: 0)

Code Examples

Use these examples to interact with the API using popular tools and languages.

Create a Resume (JavaScript)

const response = await fetch('https://api.resumebuilder.com/v1/resumes', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    userId: 'user123',
    templateId: 'professional-template',
    personalInfo: {
      name: 'Jane Smith',
      email: 'jane.smith@example.com',
      phone: '+1-555-9876',
      address: '123 Main St, Boston, MA'
    },
    experience: [
      {
        company: 'Tech Solutions',
        position: 'Software Engineer',
        startDate: '2021-06',
        endDate: '2025-08',
        description: 'Developed scalable web applications using ASP.NET Core.'
      }
    ],
    education: [
      {
        institution: 'State University',
        degree: 'B.S. Computer Science',
        field: 'Computer Science',
        startDate: '2017-09',
        endDate: '2021-05'
      }
    ],
    skills: ['C#', 'ASP.NET Core', 'JavaScript']
  })
});

const resume = await response.json();
console.log(resume);

Export Resume to PDF (Python)

import requests

url = "https://api.resumebuilder.com/v1/resumes/resume123/export"
headers = {
    "Authorization": "Bearer YOUR_API_KEY",
    "Content-Type": "application/json"
}
data = {
    "format": "pdf",
    "options": {
        "theme": "modern",
        "fontSize": 11,
        "margins": "narrow"
    }
}

response = requests.post(url, headers=headers, json=data)
if response.ok:
    pdf_url = response.json()["data"]["downloadUrl"]
    print(pdf_url)

Get User Resumes (cURL)

curl -X GET "https://api.resumebuilder.com/v1/users/user123/resumes?limit=5&offset=0" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type": "application/json"
